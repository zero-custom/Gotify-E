# i18n — 多语言 UI 层

网关在代理 Gotify HTML 页面时，注入客户端 i18n 层。该层将 Gotify 前端 UI 及其相对时间戳翻译为浏览器偏好的语言，无需修改后端。

## 架构

```
proxy.py:inject_i18n()          服务端：注入 <script> 标签
    │
    ├─ /_gateway/i18n.js         引擎 — 语言检测、文本遍历、
    │                              相对时间翻译（无状态）
    ├─ /_gateway/enhance.js       增强层 — 品牌标记、布局、时间切换、
    │                              引导 i18n（有状态）
    └─ /_gateway/lang/{lang}.js  静态语言数据 — `__LOCALE_MAP` 数组
                                    （纯数据，无逻辑）
```

**i18n.js** 提供被动引擎（检测、遍历、替换）。**enhance.js** 作为消费者，同时处理品牌标记、时间切换按钮和布局 CSS 等功能。翻译文件（`zh_CN.js`、`fr.js` 等）是纯 `[en, translated]` 二元组数组。

## 语言检测（优先级顺序）

1. **`?lang=` 查询参数** — 最高优先级，例如 `?lang=fr`
2. **`localStorage` 键 `gotify-lang`** — 跨会话持久化
3. **`navigator.language` / `navigator.userLanguage`** — 浏览器偏好

### 浏览器自动检测

引擎将语言地区代码归一化为翻译文件名：

| 浏览器地区 | 翻译文件 |
|---|---|
| `zh`、`zh-CN`、`zh-SG`、`zh-Hans` | `zh_CN.js` |
| `fr`、`fr-FR`、`fr-CA`、`fr-CH`、`fr-BE` | `fr.js` |
| `de`、`de-DE`、`de-AT`、`de-CH` | `de.js` |
| `es`、`es-ES`、`es-MX`、`es-AR` | `es.js` |
| `pt`、`pt-PT`、`pt-BR` | `pt.js` |
| `ru`、`ru-RU` | `ru.js` |
| `it`、`it-IT`、`it-CH` | `it.js` |
| `ko`、`ko-KR` | `ko.js` |
| `ja`、`ja-JP` | `ja.js` |
| 其他 | 回退至英语（不翻译） |

## 翻译机制

两个独立的 Pass：

### Pass 1 — 静态 UI 文本（`__LOCALE_MAP`）

每个语言文件暴露 `window.__LOCALE_MAP`，即 `[英文, 翻译]` 二元组数组：

```js
window.__LOCALE_MAP = [
  ['Delete All', '删除全部'],
  ['Push Message', '推送消息'],
  // … 每种语言 180 条
];
```

`walkReplace()` 遍历 DOM 文本节点，使用 locale map 执行整词替换。条目按长度降序排列以防止部分匹配。仅修改 `SCRIPT`、`STYLE`、`INPUT`、`TEXTAREA` 和 `.notranslate` 元素之外的文本。

### Pass 2 — 相对时间（`Intl.RelativeTimeFormat`）

`i18n.js` 解析 Gotify 的英语相对时间字符串（由 `react-timeago` 以 `locale:'en'` 产生），通过 `Intl.RelativeTimeFormat` 转换。

## enhance.js — 其他功能

| 功能 | 文件 | 说明 |
|---|---|---|
| **布局** | `enhance.js:injectLayoutCSS()` | 消息列表自适应宽屏，内容区宽度从 700px 扩展至视口 50% |
| **时间切换** | `enhance.js:injectTimeToggle()` | 相对/绝对时间切换按钮，持久化至 `localStorage`，按钮标签自动切换中/英 |

## 新增语言

1. 创建 `app/_gateway/lang/{code}.js`，使用相同的 `window.__LOCALE_MAP` 格式（从现有文件提取 180 条）
2. 在 `i18n.js:NORM` 对象中添加归一化条目
3. 在 `i18n.js:detectLang()` 的 `navigator.language` 块中添加浏览器检测分支

## 静态文件服务

网关将 `app/_gateway/` 目录挂载为 `/_gateway/` 静态目录：

```python
app.mount("/_gateway", StaticFiles(directory=str(_GATEWAY_DIR)), name="gateway")
```

`app/_gateway/lang/` 下的翻译文件由 `i18n.js:activate()` 动态请求加载：
- 加载成功：`__LOCALE_MAP` 填充、`Intl.RelativeTimeFormat` 创建、`walkReplace()` 执行
- 404 时：静默回退至英语，清除 `gotify-lang` 的 localStorage 键
